# AniStream

AniStream is a lightweight Android WebView application for streaming anime, built with Jetpack Compose.

## Features

- **Full-Screen Experience:** Optimized for edge-to-edge display with system bar handling.
- **High Performance:** Hardware-accelerated WebView with optimized caching and rendering.
- **Smooth Navigation:** In-app back button handling and hidden scrollbars for a native feel.
- **Modern Tech Stack:** Built with Kotlin, Jetpack Compose, and Material 3.
- **Persistent Session:** Remembers your scroll position and state even if the app is minimized.

## Getting Started

### Prerequisites

- Android Studio Koala or newer
- JDK 17+
- Android SDK 35+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/AniStream.git
   ```
2. Open the project in Android Studio.
3. Build and run the app on your device or emulator.

## Permissions

- `INTERNET`: To load the streaming site.
- `ACCESS_NETWORK_STATE`: To check connectivity.
- `WAKE_LOCK`: To prevent screen sleep during video playback.
